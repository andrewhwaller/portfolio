Portfolio

Focushift

FOCUSHIFT is an Omnifocus-inspired task and project management application. FOCUSHIFT includes Partnerships, a user-to-user project collaboration features that allows users to share their projects. Features login with Facebook OAuth2, multiple has_many :through associations, validations with error messages in views, and nested RESTful resources/routes. Implements CI/CD using CircleCI.

Technologies used:

Ruby on Rails, JavaScript, Bootstrap 4, Devise, OAuth2, Facebook API, ActionMailer, PostgreSQL, CircleCI, Heroku.

Espresserhead

ESPRESSERHEAD is a Lynchian coffee tracker application built in Sinatra. ESPRESSERHEAD takes stylistic and aesthetic cues from the work of filmmaker David Lynch while allowing users to keep track of one of David's favorite things: coffee!

Technologies used:

Sinatra, Bootstrap 4, Bcrypt, ActiveRecord, PostgreSQL, Heroku.

Deedveloper CLI Application

Deedveloper is an Indeed job search engine in Ruby. Deedveloper scrapes data directly from Indeed.com and allows users to search Indeed listings with a general keyword search as well as parameters such as location, search radius, and salary range, all without utilizing the Indeed API.

Technologies used:

Ruby, Nokogiri
